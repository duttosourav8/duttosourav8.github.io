Physics-Informed Machine Learning for Quantum-Network Routing

Developed a reproducible quantum-network routing framework using an eight-node simulation built with the SeQUeNCe quantum-network simulator. The project predicts whether candidate routes can satisfy end-to-end entanglement requests and evaluates route-selection performance on a structurally different external network topology.

The machine-learning pipeline uses physics-informed features such as fibre loss, hop count, bottleneck link success, deadline margin, fidelity margin, memory availability, effective efficiency, and coherence time. Seven classical classifiers were evaluated using leakage-safe nested group cross-validation.

The selected logistic regression model achieved an external-topology ROC–AUC of 0.950, an F1-score of 0.905, and a balanced accuracy of 0.854. When used as a routing policy, it reduced mean route-selection regret by approximately 46.5% compared with uniform random routing.

The learned policy was also compared with minimum-hop, minimum-fibre-loss, maximum-bottleneck-success, random, and oracle routing policies. It significantly outperformed random selection and achieved performance comparable to the strongest deterministic routing heuristics.

Quantum networking, SeQUeNCe, machine learning, nested cross-validation, routing optimization, SHAP, LIME
